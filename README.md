# FbInsightsWebDataConnector

## Install

```bash
npm install
```

## Development

```bash
npm start
```

## Production

### Deploy

```bash
cp .env.sample .env
```

Then, Set up the envs for yourself.

```bash
npm run deploy
```

It automatically publish the app to S3.

## Test

```bash
npm run test
```
